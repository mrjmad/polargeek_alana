# Introduction

Les [polargeek](http://polar-geek.org/). Au départ, il y a eu une nouvelle de Thibault Jouannic. Elle fut suivi d'un texte de Bruno Bord. Le mouvement était lancé, et je tentais d'y contribuer
avec mes modestes moyens en publiant deux textes sur mon blog. Toutefois la publication sur un blog n'est pas satisfaisante. Les commentaires permettent
d'échanger, mais de manière limité. Et pas de possibilité pour un lecteur de contribuer, faire une correction, proposer une amélioration...

L'idée derrière était de passer mes textes (voir tout mon blog) sur un système versionné. Mais comme toujours, quand les envies restent au stade
d'envie, rien n'avance. 
  
Et puis finalement, la découverte de gitbook m'a fait sauter le pas. Après tout, gitbook s'occupe presque de tout. 

Voici donc ici mes deux premiers polargeek. Si d'aventure, vous vouliez proposer une modification, n'hésitez pas, les Pull Requests sont là pour cela.

Bonne lecture !
